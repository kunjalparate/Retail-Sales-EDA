# Retail Sales Data Analysis - EDA Project

## Objective
Perform Exploratory Data Analysis (EDA) on Superstore Sales Dataset to:
- Understand sales patterns
- Detect anomalies
- Generate business insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Tasks Performed
- Data Loading
- Data Profiling
- Missing Value Analysis
- Univariate & Bivariate Analysis
- Correlation Analysis
- Pivot Tables
- Pairplot Exploration
- Anomaly Detection
- Business Insights

## Dataset
Superstore Sales Dataset

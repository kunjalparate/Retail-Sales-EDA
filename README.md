# Retail Sales Data Analysis - EDA Project

## Objective
Perform Exploratory Data Analysis (EDA) on the Superstore Sales Dataset to:
- Understand sales patterns
- Detect anomalies
- Generate business insights

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Tasks Performed
- Data Loading & Overview
- Data Profiling
- Missing Value Analysis
- Univariate Analysis
- Bivariate Analysis
- Pivot Table Analysis
- Correlation Analysis
- Pairplot Exploration
- Anomaly Detection
- Insight Generation

---

## Dataset Used
Superstore Sales Dataset (Sales Forecasting)

Dataset Link:
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting

---

## Project Files
- `Kunjal Parate Batch 2 - 2615891.ipynb`
- `superstore.csv`

---

## Key Insights
- Office Supplies and Technology categories generate strong sales.
- Certain regions consistently outperform others.
- A few products contribute heavily to total sales.

---

## Conclusion
This project helped analyze retail sales data using EDA techniques and generated meaningful business insights for decision-making.
